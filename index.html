Chronologle

Arrange events from earliest (top) to latest (bottom)

How to Play

Goal: Arrange the events in chronological order, earliest at the top.

Drag events to reorder them. On mobile, tap and hold to drag.

Draw more events for a higher max score, but they're harder to place. Once you draw an event, you're stuck with it — choose wisely!

Scoring: Getting an event in the exact right position earns big points. Being close earns much less — so precision matters!

🟩 Correct positionN points (N = events attempted)
🟨 Off by 1N − 2 points
🟧 Off by 2N − 4 points
🟥 Off by 3+−1 point

Example — is drawing a 6th event worth it?
Say you have 5 events all correct = 25 pts (5×5). You draw a 6th. Now each correct event is worth 6 instead of 5, and max jumps to 36.

ResultScore
😎Nail it — all 6 correct36 (+11)
🙂New event off by 1, bumps 1 other32 (+7)
😐New event off by 1, bumps 2 others30 (+5)
💀New event off by 3, bumps 3 others23 (−2)

The reward is huge if you're confident, but one bad placement can cascade and cost you. Choose wisely!

Hints:

  • Category — reveals the event type (e.g. sports, tech). Costs 1 point.
  • Decade — reveals the decade (e.g. 1990s). Costs N − 3 points (where N = events attempted). The more events you draw, the more expensive!

A new puzzle appears every day. Good luck!
