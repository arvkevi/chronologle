How to Play

Goal: Arrange the events in chronological order, earliest at the top.

Drag events to reorder them. On mobile, tap and hold to drag.

Add events for a higher score — each correct answer is worth more when you attempt more events. But be careful: more events means it's harder to get them all right!

Scoring: Each event earns up to N points (where N = events attempted). You lose 1 point for each position you're off by. Being close still counts! 🟩 = perfect, 🟨 = off by 1, 🟧 = off by 2, 🟥 = off by 3+.

Hints: Tap the ? on any event to reveal its category — but each hint costs 1 point from your final score.

A new puzzle appears every day. Good luck!
